Objective
The objective of this project is to develop a Library Management System using Python that helps in managing library operations such as:
Adding and managing books
Managing members
Issuing and returning books
Calculating fines for late returns
Maintaining records using a database

Description
This project is a GUI-based application developed using Python Tkinter for the interface and SQLite for the database.
The system supports two types of users:
Admin
Manage books (add, update, delete)
Manage memberships
View reports
User
Search books
Issue books
Return books
Pay fines

The system ensures:
Book availability checking
Membership validation
Automatic fine calculation (₹5 per day)
Secure login system

Library Management System is a Python-based application developed using Tkinter and SQLite to automate library operations. It helps manage books, memberships, issue and return processes, fine calculation, and report generation efficiently. The system provides secure admin and user login access, making library management faster, easier, and more organized.
